Address book. Privacy policy

The application does not collect and does not transmit user information.

The application needs the permission "READ_CONTACTS" for import contacts from the phone to the application.

The application needs the permission "CALL_PHONE" for be able for making calls.

The application needs the permission "SEND_SMS" for be able to send SMS.

The application needs permission “READ_CALL_LOG” for populate the call list.

The application needs permissions “WRITE_EXTERNAL_STORAGE” and "READ_EXTERNAL_STORAGE" for working with avatars of contacts.

The application is not an online service. Internet is spent only for downloading banners
